Key Design Decisions

A

Centered Floating Modal

Modal appears in center of screen (like Spotlight). Dimmed background. Best for keyboard-driven search.

B

Top Bar Dropdown

Search bar appears at top of main window, results drop down below it. More integrated with the app layout.

C

Sidebar Panel

Search results appear in the main content area when "Search" is selected in sidebar. More traditional, less modal.

Result Types

1

Commits

Search commit messages, authors, and hashes. Show commit title, hash, author, time, and branch.

2

Files

Search file names and paths. Show file name, directory path, and git status (modified/staged/untracked).

3

Branches

Search local and remote branch names. Show branch name, remote tracking info, and branch type.

4

Tags

Search tag names. Show tag name and associated commit info.
